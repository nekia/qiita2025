プライバシーポリシー

LINE Message＆フォトフレーム用キオスク端末アプリ（以下「本アプリ」といいます）は、利用者のプライバシーを尊重し、 個人情報および写真データの適切な取り扱いに努めます。

1. 取得する情報

本アプリは、利用者の明示的な同意に基づき、以下の情報を取得する場合があります。

  • Google Photos API を通じて取得される写真・画像データ
  • 写真に付随する最小限のメタデータ（撮影日時等）

2. 情報の利用目的

取得した情報は、以下の目的にのみ利用されます。

  • キオスク端末等の画面上に写真・画像を表示するため
  • 本アプリの機能提供および品質向上のため

3. 情報の保存と管理

本アプリは、写真・画像データを永続的に保存することを目的としていません。 表示に必要な範囲で一時的に処理される場合を除き、第三者が閲覧可能な形で保存されることはありません。

4. 第三者提供について

本アプリは、法令に基づく場合を除き、取得した情報を第三者に提供することはありません。

5. Google API の利用について

本アプリは Google Photos API を利用しており、 Google API Services User Data Policy に準拠してユーザーデータを取り扱います。

取得したデータは、Google が定めるポリシーに従い、許可された範囲内でのみ使用されます。

6. 利用者の選択と権利

利用者は、Google アカウントの設定画面から、 いつでも本アプリへのアクセス権限を取り消すことができます。

7. プライバシーポリシーの変更

本ポリシーは、必要に応じて予告なく変更されることがあります。 変更後の内容は、本ページに掲載された時点で効力を生じるものとします。
